SQLжіЁе…ҘвҖҰвҖҰзҗҶи®әдёҠйңҖиҰҒдёҖе®ҡзҡ„PHPе’ҢMySQLеҹәзЎҖпјҢз„¶иҖҢдёӨдёӘжҲ‘йғҪиҝҳжІЎеӯҰпјҲеҲ«йӘӮдәҶпјүпјҢжүҖд»ҘжіЁе®ҡжҳҜеӣ°йҡҫйҮҚйҮҚпјҢеӯҰд№ ж•ҲжһңеҰӮдҪ•д№ҹдёҚж•ўдҝқиҜҒгҖӮжҡ‘еҒҮзҡ„ж—¶еҖҷеҗ¬дәҶеҗ¬еӯҰж ЎеӨҸд»ӨиҗҘпјҢж„ҹи§үжңүдәҶзӮ№зңүзӣ®пјҢз„¶иҖҢиҝҳжҳҜеҶҷдёҚеҮәжқҘйўҳзӣ®пјҲSQLиҜӯеҸҘеӨӘй•ҝдәҶи®°дёҚдҪҸвҖҰвҖҰпјүгҖӮж— и®әеҰӮдҪ•пјҢиҝҳжҳҜе…ҲејҖе§Ӣеҗ§гҖӮ

еӣҫ1-5

иҝҷдёӘиЎҢдёәеҹәжң¬еҸҜд»ҘеҲӨж–ӯиҜҘжіЁе…ҘзӮ№дёәж•°еӯ—еһӢжіЁе…ҘпјҢиЎЁзҺ°дёә$_GET['id']йҷ„иҝ‘жІЎжңүеј•еҸ·еҢ…иЈ№гҖӮиҝҷж—¶жҲ‘д»¬еҸҜд»ҘзӣҙжҺҘиҫ“е…ҘSQLжҹҘиҜўиҜӯеҸҘжқҘе№Іжү°жӯЈеёёзҡ„жҹҘиҜўгҖӮ

SELECT title, content FROM wp_news WHERE id = 1 UNION SELECT user, pwd FROM wp_user

иҝҷдёӘSQLиҜӯеҸҘзҡ„дҪңз”ЁжҳҜжҹҘиҜўж–°й—»иЎЁдёӯid=1ж—¶еҜ№еә”иЎҢзҡ„titleе’Ңcontentеӯ—ж®өзҡ„ж•°жҚ®пјҢе№¶дё”иҒ”еҗҲжҹҘиҜўз”ЁжҲ·иЎЁдёӯзҡ„userгҖҒpwdгҖӮunionе…ій”®еӯ—дјҡеҜ№дёӨдёӘз»“жһңйӣҶиҝӣиЎҢе№¶иЎҢжҹҘиҜўгҖӮ

еӣҫ1-6

дҪҶжҳҜеҰӮеӣҫ1-6жүҖзӨәпјҢзҪ‘йЎөе№¶жІЎжңүжҢүз…§йў„жңҹжҳҫзӨәз”ЁжҲ·е’ҢеҜҶз Ғзҡ„еҶ…е®№пјҢдәӢе®һдёҠMySQLзЎ®е®һжҹҘиҜўеҮәдәҶдёӨжқЎи®°еҪ•пјҢдҪҶеӣ дёәPHPйҷҗеҲ¶иҜҘйЎөйқўеҸӘиғҪжҳҫзӨәдёҖжқЎеҶ…е®№пјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒе°ҶиҙҰеҸ·еҜҶз Ғзҡ„и®°еҪ•жҳҫзӨәеңЁжҹҘиҜўз»“жһңз¬¬дёҖиЎҢгҖӮжңүеҮ з§Қе®һзҺ°ж–№ејҸпјҢжҜ”еҰӮеңЁеҺҹжңүж•°жҚ®еҗҺеҠ дёҠlimit 1,1еҸӮж•°пјҲжҳҫзӨәжҹҘиҜўз»“жһңеҗҺзҡ„з¬¬дәҢжқЎи®°еҪ•пјүгҖӮеҸҲжҲ–иҖ…жҢҮе®ҡid=-1жҲ–иҖ…дёҖдёӘеҫҲеӨ§зҡ„еҖјпјҢз¬¬дёҖиЎҢи®°еҪ•ж— жі•иў«жҹҘиҜўеҲ°пјҢиҝҷж ·з»“жһңе°ұеҸҜд»ҘеҸӘжҳҫзӨәз¬¬дәҢжқЎи®°еҪ•дәҶгҖӮ

йӮЈд№ҲжҲ‘д»¬и®ҝй—®http://192.168.20.133/sql1.php?id=-1 union select user, pwd from wp_userжҲҗеҠҹеҫ—еҲ°дәҶеӣҫ1-7жүҖзӨәзҡ„з”ЁжҲ·иЎЁзҡ„иҙҰеҸ·е’ҢеҜҶз ҒгҖӮ

еӣҫ1-7

д»ҘдёҠиҝҷз§ҚйҖҡиҝҮUNIONиҜӯеҸҘе°Ҷж•°жҚ®жҳҫзӨәеҲ°йЎөйқўдёҠзҡ„жіЁе…Ҙж–№жі•дёәUNIONжіЁе…Ҙ/иҒ”еҗҲжіЁе…ҘгҖӮжҳҜжңҖдёәз®ҖеҚ•д№ҹжҳҜжңҖеҹәзЎҖзҡ„жіЁе…Ҙж–№жі•пјҢеӣ дёәиҜҘж•°жҚ®еә“жІЎжңүи®ҫзҪ®д»»дҪ•е…ідәҺж•°жҚ®зҡ„иҝҮж»Өж–№ејҸгҖӮжҲ‘д»¬еҗҺз»ӯеӯҰд№ зҡ„жіЁе…Ҙж–№жі•еӨ§еӨҡд№ҹеҹәдәҺиҝҷз§ҚжҖқи·ҜпјҢдёҚиҝҮиҰҒе°қиҜ•з»•иҝҮжүҖи®ҫзҪ®зҡ„иҝҮж»ӨпјҲд№ҹи®ёжҳҜеҗ§пјүгҖӮ

дҪҶжҳҜд»ҘдёҠзҡ„дҫӢеӯҗжҳҜй»ҳи®ӨжҲ‘д»¬е·Із»ҸзҹҘйҒ“дәҶж•°жҚ®еә“з»“жһ„пјҢеңЁжөӢиҜ•жғ…еҶөдёӢпјҢжҲ‘д»¬иҰҒеҰӮдҪ•зҹҘйҒ“ж•°жҚ®иЎЁзҡ„еӯ—ж®өеҗҚpwdе’ҢиЎЁеҗҚwp_userе‘ўпјҹ

зӯ”жЎҲжҳҜinformation_schemaгҖӮMySQL5.0зүҲжң¬еҗҺпјҢй»ҳи®ӨиҮӘеёҰдёҖдёӘж•°жҚ®еә“information_schemaпјҢеӮЁеӯҳзқҖMySQLзҡ„жүҖжңүж•°жҚ®еә“еҗҚгҖҒиЎЁеҗҚгҖҒеӯ—ж®өеҗҚгҖӮиҷҪз„¶еј•е…ҘиҝҷдёӘеә“жҳҜж–№дҫҝж•°жҚ®еә“дҝЎжҒҜзҡ„жҹҘиҜўпјҢдҪҶжҳҜе®ўи§ӮдёҠеҚҙеӨ§еӨ§ж–№дҫҝдәҶSQLжіЁе…ҘпјҲд№җпјүгҖӮ

жҺҘдёӢжқҘжј”зӨәе®һжҲҳзҡ„ж“ҚдҪңиҝҮзЁӢгҖӮ

еҒҮи®ҫжҲ‘д»¬е№¶дёҚзҹҘйҒ“ж•°жҚ®еә“зҡ„зӣёе…ідҝЎжҒҜпјҢйҰ–е…ҲйҖҡиҝҮid=2е’Ңid=3-1еӣһжҳҫдёҖиҮҙеҲӨж–ӯиҝҷеӯҳеңЁдёҖдёӘж•°еӯ—еһӢжіЁе…ҘпјҢз„¶еҗҺйҖҡиҝҮиҒ”еҗҲжҹҘиҜўпјҢжҹҘеҲ°жң¬ж•°жҚ®еә“е’Ңе…¶д»–жүҖжңүиЎЁеҗҚгҖӮи®ҝй—®http://192.168.20.133/sql1.php?id=-1 union select 1,group_concat(table_name) from information_schema.tables where table_schema=database()гҖӮз»“жһңи§Ғеӣҫ1-8гҖӮ

еӣҫ1-8

и®©жҲ‘д»¬зңӢзңӢиҝҷд№Ҳй•ҝдёҖдёІзҺ©ж„ҸеҲ°еә•жү§иЎҢдәҶдёӘе•ҘгҖӮinformation_schemaеҢ…еҗ«дәҶMySQLдёӯжүҖжңүзҡ„ж•°жҚ®еә“еҗҚгҖҒиЎЁеҗҚгҖҒеӯ—ж®өеҗҚпјҢе…¶дёӯзҡ„tableиЎЁе°ұжҳҜеӮЁеӯҳж•°жҚ®еә“жүҖжңүиЎЁеҗҚзҡ„дёҖдёӘиЎЁпјҢиҖҢtable_nameжҳҜtableиЎЁзҡ„иЎЁеҗҚеӯ—ж®өпјҢеҗҺйқўи·ҹжүҖжңүиЎЁзҡ„еҗҚеӯ—гҖӮtableиЎЁдёӯиҝҳеҢ…еҗ«ж•°жҚ®еә“еҗҚеӯ—ж®өtable_schemaпјҢdatabase()еҮҪж•°иҝ”еӣһзҡ„еҶ…е®№жҳҜеҪ“еүҚж•°жҚ®еә“зҡ„еҗҚз§°пјҢgroup_concat()еҮҪж•°еҲҷжҳҜз”ЁвҖң,вҖқжқҘиҒ”еҗҲеӨҡиЎҢи®°еҪ•зҡ„еҮҪж•°гҖӮ

д№ҹе°ұжҳҜиҜҙпјҢиҜҘиҜӯеҸҘеҸҜд»ҘжҹҘиҜўж•°жҚ®еә“дёӯжүҖжңүиЎЁеҗҚе№¶жҳҫзӨәеңЁдёҖдёӘеӯ—ж®өдёӯгҖӮеҰӮдёӢеӣҫ1-9жүҖзӨәгҖӮ

еӣҫ1-9

жӣҙиҜҰз»Ҷзҡ„ChatGPTи§ЈйҮҠйҷ„еңЁдёӢж–№гҖӮ

еҪ“и§ЈйҮҠиҝҷжқЎSQLиҜӯеҸҘдёӯзҡ„е…ій”®еӯ—ж—¶пјҢжҲ‘дјҡдёҖдёҖиҜҙжҳҺе®ғд»¬зҡ„дҪңз”Ёпјҡ

  1. union: еңЁSQLдёӯпјҢUNIONж“ҚдҪңз¬Ұз”ЁдәҺеҗҲе№¶дёӨдёӘжҲ–еӨҡдёӘSELECTиҜӯеҸҘзҡ„з»“жһңйӣҶгҖӮе®ғе°ҶеӨҡдёӘжҹҘиҜўзҡ„з»“жһңеҗҲе№¶еңЁдёҖиө·пјҢе№¶д»ҘдёҖдёӘеҚ•дёҖзҡ„з»“жһңйӣҶиҝ”еӣһгҖӮ
  2. select: SELECTе…ій”®еӯ—з”ЁдәҺд»Һж•°жҚ®еә“иЎЁдёӯжЈҖзҙўж•°жҚ®гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢSELECTз”ЁдәҺйҖүжӢ©жҹҘиҜўз»“жһңдёӯйңҖиҰҒжҳҫзӨәзҡ„еҲ—гҖӮ
  3. 1: иҝҷжҳҜдёҖдёӘеёёйҮҸеҖјпјҢиЎЁзӨәз¬¬дёҖдёӘжҹҘиҜўз»“жһңйӣҶдёӯзҡ„дёҖеҲ—гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢе®ғз”ЁдҪңеҚ дҪҚз¬ҰпјҢеӣ дёәUNIONж“ҚдҪңиҰҒжұӮжүҖжңүжҹҘиҜўе…·жңүзӣёеҗҢж•°йҮҸе’Ңзұ»еһӢзҡ„еҲ—гҖӮ
  4. group_concat: GROUP_CONCATжҳҜMySQLдёӯзҡ„иҒҡеҗҲеҮҪж•°д№ӢдёҖгҖӮе®ғз”ЁдәҺе°ҶеӨҡдёӘиЎҢзҡ„жҹҗдёҖеҲ—еҖјиҝһжҺҘжҲҗдёҖдёӘеҚ•зӢ¬зҡ„еӯ—з¬ҰдёІгҖӮеңЁиҝҷйҮҢпјҢе®ғз”ЁдәҺе°Ҷtable_nameеҲ—дёӯзҡ„иЎЁеҗҚиҝһжҺҘжҲҗдёҖдёӘеӯ—з¬ҰдёІгҖӮ
  5. table_name: table_nameжҳҜinformation_schema.tablesи§Ҷеӣҫдёӯзҡ„дёҖдёӘеҲ—еҗҚгҖӮиҜҘеҲ—еҢ…еҗ«дәҶж•°жҚ®еә“дёӯзҡ„иЎЁеҗҚдҝЎжҒҜгҖӮ
  6. information_schema.tables: information_schemaжҳҜMySQLзі»з»ҹж•°жҚ®еә“д№ӢдёҖпјҢе…¶дёӯеҢ…еҗ«е…ідәҺж•°жҚ®еә“гҖҒиЎЁгҖҒеҲ—е’Ңе…¶д»–ж•°жҚ®еә“еҜ№иұЎзҡ„е…ғж•°жҚ®дҝЎжҒҜгҖӮtablesжҳҜinformation_schemaдёӯзҡ„дёҖдёӘиЎЁпјҢеҢ…еҗ«жңүе…іж•°жҚ®еә“дёӯиЎЁзҡ„дҝЎжҒҜгҖӮ
  7. where: WHEREе…ій”®еӯ—з”ЁдәҺжҢҮе®ҡжҹҘиҜўзҡ„жқЎд»¶гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢwhere table_schema=database()з”ЁдәҺиҝҮж»ӨеҸӘиҝ”еӣһеҪ“еүҚж•°жҚ®еә“дёӯзҡ„иЎЁгҖӮ

йҖҡиҝҮз»“еҗҲдҪҝз”Ёиҝҷдәӣе…ій”®еӯ—е’ҢиҜӯеҸҘпјҢж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁжіЁе…ҘжјҸжҙһжү§иЎҢжҒ¶ж„ҸжҹҘиҜўпјҢе№¶д»Һinformation_schema.tablesиЎЁдёӯжҸҗеҸ–иЎЁеҗҚдҝЎжҒҜгҖӮ

еҗҢзҗҶпјҢйҖҡиҝҮcolumnsиЎЁдёӯзҡ„column_nameжҹҘиҜўеҮәзҡ„еҶ…е®№еҚідёәwp_userдёӯзҡ„еӯ—ж®өеҗҚпјҢи®ҝй—®http://192.168.20.133/sql1.phpпјҹid=-1 union select 1 ,group_concat (column_name) from information_schema.columns where table_name='wp_user'пјҢеҸҜд»Ҙеҫ—еҲ°еҜ№еә”зҡ„еӯ—ж®өеҗҚгҖӮи§Ғеӣҫ1-10гҖӮ

еӣҫ1-10

иҮіжӯӨпјҢз¬¬дёҖдёӘдҫӢеӯҗз»“жқҹгҖӮж•°еӯ—еһӢжіЁе…Ҙзҡ„е…ій”®еңЁдәҺжүҫеҲ°иҫ“е…Ҙзҡ„еҸӮж•°зӮ№пјҢз„¶еҗҺйҖҡиҝҮеҠ еҮҸд№ҳйҷӨзӯүиҝҗз®—еҲӨж–ӯеҸӮж•°йҷ„иҝ‘жңүж— еј•еҸ·еҢ…иЈ№пјҢеҶҚйҖҡиҝҮдёҖдәӣеёёи§Ғзҡ„ж”»еҮ»жүӢж®өпјҲжҜ”еҰӮиҒ”еҗҲжіЁе…ҘпјҢеңЁжӯӨд№ӢеүҚе…Ҳе°қиҜ•иҺ·еҫ—иЎЁеҗҚпјүпјҢиҺ·еҸ–ж•°жҚ®еә“зҡ„ж•Ҹж„ҹдҝЎжҒҜгҖӮ
